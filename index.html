Apresentação e propósito

Olá, meu nome é Leandro Ferreira Das Neves, tenho 18 anos e resido em Americana, São Paulo. Decidi criar esse site para armazenar e disponibilizar código-fonte e arquivos de minha autoria, com intuito de demonstrar minhas habilidades técnicas e minha evolução na programação.

Projetos

Discovery Code

Este projeto foi desenvolvido no Trabalho de conclusão de curso do SENAI. Trata-se de um sistema onde usuários cadastram perguntas relacionadas a programação e são respondidos por outros usuários.

Autores: Leandro Neves, Isabella Camargo e Kaiky Campos.

Projeto disponível em https://github.com/leandrofneves/Discovery-Code

Controle de produtos

Sistema que possibilita cadastrar, editar e excluir um produto do banco de dados. Além de trabalhar com consultas no banco de dados, esse projeto nos mostra o uso correto do Inner Join.

Autor: Leandro Neves.

Projeto disponível em https://github.com/leandrofneves/Controle-Produtos
